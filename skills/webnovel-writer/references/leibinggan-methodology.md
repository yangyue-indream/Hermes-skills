# 雷饼干写小说赚钱方法论

> 来源：蒸馏抖音博主「雷饼干」（2.7万粉，7万赞）全部37个视频（2026.05.28-06.23）
> 作者背景：5年网文老扑街，最高单本稿费20万，3次短剧改编，从月薪三千到稿费20万

## 一、核心心法

### 1. "抄"字诀
写小说月入过万的方法是"抄"——抄核心卖点、抄爽点、融合其他火书。
- 抄核心卖点（不是逐字抄，是抄"为什么好看"）
- 抄爽点结构
- 融合多本火书（A 书的框架 + B 书的人设 + C 书的节奏）
- 底线：完全抄袭不行，要"致敬"不要"照搬"

### 2. "唯一焚决"（解决写不出来）
- 多看：扫榜、拆书、积累素材
- 多想：分析为什么好看、爽点在哪
- 多写：动笔最重要，先完成再完美
- "不要纠结，干就完了" — 3小时写2000字的问题在于纠结，不是能力

### 3. 正反馈驱动
- 写小说性价比高，但需要正反馈循环
- 收入模式：持续更新 → 读者积累 → 收入增长
- 思维模式从"我要写"变成"我要赚"
- 写小说不要卖弄文笔，越在意文笔越扑街

## 二、新人成长五阶段

1. **萌新自嗨期** → 写自己喜欢的，培养习惯
2. **扫榜跟风期** → 大量阅读热门，模仿学习
3. **步入正轨期** → 找到自己的节奏和方向
4. **风格成型期** → 形成独特风格
5. **证道成神期** → 持续产出的高手

## 三、新手实操路径

1. 先动笔，别想太多 — "90%的人都能写小说，只要肯坚持"
2. 不要纠结文笔，越在意越扑街
3. 新人拆书方法：扫榜 → 找热门 → 分析结构 → 模仿
4. 雷饼干自己：5年 → N本扑街 → 最高单本20万 → 3次短剧改编

## 四、AI + 写小说

- AI 写小说能赚钱
- AI 是工具，核心还是懂网文套路
- 用 AI 辅助拆书、生成灵感、提速
- 有专门的 AI 写小说赚钱全流程教学

## 五、平台与实操

- 番茄小说是新人的主要推荐平台（新人有流量扶持）
- 番茄有"造神计划"（有利有弊需注意）
- 新人开书全流程：注册 → 选类型 → 写开头 → 持续更新 → 看数据
- 追读率是最重要的数据指标

## 六、常见避坑

| 坑 | 解法 |
|-----|------|
| 卖弄文笔 | 网文看的是爽点不是文笔 |
| 纠结纠结 | 干就完了，别纠结 |
| 错误创新 | 先模仿再创新，别一上来就搞原创 |
| 全职太早 | 先用业余时间验证能赚钱再全职 |
| 反复扑街 | 分析原因：是选题不对还是节奏不好 |
| 没正反馈 | 坚持写，数据会慢慢起来 |
